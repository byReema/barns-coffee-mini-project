# Barn's Coffee Branch Performance Manager

A simple Python project to manage and analyze branch performance for Barn's Coffee.

## Features

- Create branch objects from raw data
- Validate branch data
- Analyze branch performance
- Show high-risk branches
- Search branches by city
- Add new branches
- Generate full performance reports
- Save reports as text files

## Files

- `main.py` → runs the project
- `branch.py` → contains classes and functions
- `README.md` → project description

## How to Run

1. Make sure Python is installed
2. Open terminal in the project folder
3. Run:

```bash
python main.py
